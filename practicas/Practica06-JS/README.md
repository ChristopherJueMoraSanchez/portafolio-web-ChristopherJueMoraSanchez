CALCULADORA JS

*Objetivo.*

Usando los conocimientos adquiridos hasta ahora, crear una calculadora la cual tenga diferentes funciones, como puede ser la suma, resta, multiplicación y división.
Esto demuestra que no solo puedes hacer algo visual, si no, tambien algo funcional.

*Tecnologias usadas.*

Visual Studio Code
Github
Git Code
HTML5
CSS3
JS

*Instrucciones.*

1. Localizar el archivo index.html
2. Dar doble clic para abrirlo.
3. Visualizar la pagina.
4. Colocar los números en los espacios correspondientes.
5. Elegir el tipo de proceso.
6. Ver el resultado.
