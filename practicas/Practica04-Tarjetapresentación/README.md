TARJETA DE PRESENTACIÓN

*Objetivo*

Mediante el uso de CSS y HTML hacer una pagina que es una tarjeta de presentación. Teniendo varios botones que tienen algun efecto al pasar el mouse por encima, además de insertar un link en uno de esos botones.

*Tecnologias usadas.*

Visual Studio Code
Github
Git Code
HTML
CSS

*Instrucciones.*

1. Localizar el archivo index.html
2. Darle doble clic.
3. Visualizar la pagina web.
