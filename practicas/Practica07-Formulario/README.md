FORMULARIO

*Objetivo.*

Mediante esta practica, se aprende como realizar un formulario para una pagina web, como se puede personalizar, como guardar la información y adicional una pagina para visualziar los datos guardados.

*Tecnologias Usadas.*

Visual Studio Code
Github
Git Code
HTML5
CSS3
JS

*Instrucciones.*

1. Entrar en la carpeta InicioSesión Basico.
2. Localizar el archivo index.html
3. Darle doble clic para abrirlo.
4. Entrar a la carpeta API.
5. Localizar el archivo index.html
6. Darle doble clic para abrirlo.
7. Darle al botón de Cargar Usuarios.
8. Copiar el correo y contraseña de los usuarios de ejemplo.
9. Regresar a la pagina de index.html de InicioSesión Basico.
10. Colocar el usuario.
11. Colocar la contraseña.
12. Darle clic en el botón ingresar.
