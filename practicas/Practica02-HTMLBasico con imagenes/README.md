Saber como insertar imagenes dentro de un html, crear una lista, ordenar una lista, insertar tipografias, entre otras opciones incluidas para personalizar una pagina web con html.

*Tecnologias usadas.*

Visual Studio Code
Github
Git Code

*Instrucciones.*

1. Localizar el archivo index.html
2. Darle clic dos veces.
3. Visualizar la pagina ya ejecutada.
